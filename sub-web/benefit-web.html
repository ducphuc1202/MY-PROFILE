Benefit Website
Flaticon Nếu bạn cần tìm những icon để phục vụ cho việc code web, phần mềm hay thậm chí là photoshop, thì Flaticon là lựa chọn không thể bỏ qua. Với hàng nghìn icon được xây dựng sẵn, bạn chỉ cần truy cập và nhận ngay cho mình những icon đúng như mong muốn. Flaticon hỗ trợ cả Free và Pro. Tuy nhiên, với gói Free cũng đủ để bạn thỏa sức sáng tạo. Bản thân mình cũng dùng Flaticon cho việc code web.
Link web: https://www.flaticon.com/
Google Font Với những ai thiết kế và lập trình web, thì font-family là điều không thể thiếu. Nó góp phần làm cho website bắt mắt hơn, dễ đọc hơn, ấn tượng hơn. Ngoài việc tự tay upload font vào trình duyệt, thì bây giờ bạn chỉ cần vào Google Font, lấy mã nhúng và mã CSS cho từng loại font, như vậy là bạn đã có font mình muốn. Tuy nhiên, hiện nay, Google vẫn còn thiếu một số font chữ. Hãy nhớ chọn ngôn ngữ là Vietnammese để tiện cho việc code web nhé.
Link web: https://fonts.google.com/
Freepik Nói đến thiết kế và photoshop, thì vector , psd, icon, photo là những thứ cốt yếu. Freepik sẽ là sự lựa chọn cho bạn. Tại đây bạn có thể tìm kiếm rất nhiều mẫu vector, psd, icon. Trang web này tương tự như Flaticon, gồm cả Free và Pro. Hãy tự khám phá trang web tuyệt vời này nhé, nó cũng rất hữu ích cho việc code web
Link web: https://www.freepik.com/
Freebiesbug Khi bạn muốn thực hành cho việc code web sau khi học tập, hay thậm chí là tìm cho mình 1 giao diện web bắt mắt để tham khảo khi làm đồ án cho khách hàng, thì Freebiesbug sẽ thật sự hữu ích cho bạn. Website này cung cấp rất nhiều mẫu PSD của các đồ án. Bạn có thể download về máy, dùng phần mềm photoshop để lấy ảnh, font, màu sắc. Việc còn lại là sáng tạo và code trang web mà bạn muốn. Mình cũng đã thử làm rất nhiều giao diện từ website này
Link web: https://freebiesbug.com/
